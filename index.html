What is Virtual Reality?

  • Immersing into VR is a perception of being physically present in a non-physical world.
  • We surround user of the VR system in images, sound or other stimuli that provide perception of being surrounded in a virtual world.
  • Visual Immersion + Input Immersion

What is Web VR?

  • Experimental JavaScript API
  • Renders web content in Virtual Reality
  • You see device-agnostic VR in Web Browsers

Focus on Visual Immersion

  • Support both WebGL and HTML/CSS content
  • Allow High End web experience on browser with VR
    • CSS 3D
    • VR Browsing
    • Responsive sites
    • Game Engines
    • ASM JS

Supported on these browsers

Web VR API

  • hardwareUnitId: Unique identifier per Hardware Device
  • deviceId: Unique ID for specific sensor/device on hardware
  • deviceName: User-readable name identifying the sensor

Field of View (FOV) for VR devices

  • Configuring FOV that rendering device would use
    • setFieldOfView(left, right, znear, zfar)
  • Getting recommended and maximum FOV that device can render:
    • getRecommendedEyeFieldOfView(whichEye)
    getMaximumEyeFieldOfView(whichEye)
    getCurrentEyeFieldOfView(whichEye)

Way too much work!

Let's use A Frame instead

A web framework for building VR Experiences

Include it in your web page:

<script src="https://aframe.io/releases/0.3.2/aframe.min.js"></script>

Set up a scene

<a-scene></a-scene>

Create a box

<a-box color=“#EEEFDE3” height=“10” width=“4” depth=“2” position=“-10 2 -5” rotation=“0 0 45” scale=“2 0.5 3”></a-box>

Giving it a texture

<a-box color=“#EEEFDE” height=“10” width=“4” depth=“2” position=“-10 2 -5” rotation=“0 0 45” scale=“2 0.5 3” src=“texture.png”></a-box>

Leveraging Asset Management

<a-assets></a-assets>

Adding image assets

<a-assets><img id=“img1” src=“logo.png”> </a-assets>

Now let's use assets


                        <a-box color=“#EEEFDE” height=“10” width=“4” depth=“2” position=“-10 2 -5” rotation=“0 0 45” scale=“2 0.5 3” src=“#img1”> </a-box>

Adding Animations


                        <a-animation attribute="rotation" repeat="indefinite" to="0 360 0"> </a-animation>

Adding Background


                        <a-sky color=“#73f7dd”> </a-sky>

Adding Background Image


                        <a-sky src=“room.jpg”> </a-sky>

Adding some music


                        <a-entity sound=“src: #bg-music; autoplay: true; volume: 10; loop: true”> </a-entity>
